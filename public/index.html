Liendo Desarrollos

¡Hola! Mi nombre es Julián Liendo, tengo 21 años y soy estudiante de Ingeniería en Sistemas en la Universidad Tecnológica Nacional - Facultad Regional Córdoba. Me especializo en el desarrollo Web Front-end, con habilidades en HTML, CSS y JavaScript, junto a sus frameworks y herramientas. Actualmente, mi objetivo principal consiste en adquirir conocimientos para lograr ser un Desarrollador Full Stack, como así también, finalizar mis estudios universitarios con el fin de obtener un desarrollo a nivel profesional.

Curriculum

HTML Desarrollo la estructura de la página con las mejores prácticas orientadas al posicionamiento SEO de su Web, buscando así el mejor puesto en buscadores como Google. Busco una solucion simple para un rapido y eficaz mantenimiento del sitio.

CSS Interfases pensadas y maquetadas en base a conocimientos en diseño y usabilidad, atractivas como a la vez funcionales para la mejor experiencia de usuario. Utilizo herramientas como Boostrap, SASS y LESS.

JavaScript Complementando las interfaces de usuario utilizamos el lenguaje de programacion JavaScript para desarrollar animaciones que sorprendan y atraigan al usuario, trabajo con los frameworks React y Angular (TypeScript).

Bases de Datos Diseñamos, mejoramos e implementamos bases de datos adaptadas a las necesidades de su negocio. Utilizando tecnologías SQL y NoSQL en conjunto con distintos lenguajes de programación para posibilitar así adaptarnos a proyectos ya desarrollados.

Lenguajes Cada lenguaje ofrece sus grandes posibilidades, ventajas y desventajas, cada proyecto tiene sus requerimientos y en base a estos seleccionamos un lenguaje o trabajamos con el de su proyecto ya existente.

UI/UX Interfaces de Usuario y Experiencia de Usuario Diseñamos una web fácilmente utilizable para una perfecta experiencia de usuario, reforzando la imagen positiva de la marca y la conexion con el cliente mediante su pagina web.

Illustraciones Diseño de illustraciones al estilo de su marca complementando la estetica visual del producto final. Brindando acabados personalizados al mejor nivel
